
### 文本/JSON文件上传

**简介：**

支持直接上传文本内容或 JSON 数据为文件，无需本地文件，适用于生成配置文件、日志文件、JSON 数据备份等场景。

**参数说明：**

| 参数     | 类型   | 必填 | 说明                   |
| -------- | ------ | ---- | ---------------------- |
| fileName | string | 是   | 文件名(带后缀，如 hello.txt、data.json) |
| content  | string | 是   | 文本内容或 JSON 字符串 |
