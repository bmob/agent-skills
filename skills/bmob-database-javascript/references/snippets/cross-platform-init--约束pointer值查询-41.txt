
### 视频缩略图

有时候视频需要动态截取缩略图，可以使用以下接口

**请求参数**

| 参数              		| 类型   	| 必选	| 说明                                    			|
|-----------------------|-----------|-------|---------------------------------------------------|
| source 				| string 	| 是  | 视频的存储地址					|
| save_as 				| string 	| 是  | 截图保存地址					|
| point   				| string 	| 是  | 截图时间点，格式为 `HH:MM:SS`					|
