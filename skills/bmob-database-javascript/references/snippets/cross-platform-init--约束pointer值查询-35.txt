
如果多文件上传，就把`const file = Bmob.File(options.file.name, options.file);`放在for里面即可

### 小程序文件上传

**参数说明：**

| 参数     | 类型   | 必填 | 说明                   |
| -------- | ------ | ---- | ---------------------- |
| fileName | string | 是   | 文件名(带后缀)         |
| file     | Object | 是   | 相应的文本或者二进制流 |

给页面上传按钮一个点击事件，这里使用了for，支持批量上传
